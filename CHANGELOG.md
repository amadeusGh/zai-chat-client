## [0.1.0] - 2026-03-01

### Added

- Initial public package structure for `zai_chat_client`
- Async client lifecycle with session/cookie/manual-login auth modes
- Chat create/open/delete workflows
- Message sending and response tracking
- Deep Think and Web Search toggles
- Regeneration support
- Optional action pacing and colored logs
